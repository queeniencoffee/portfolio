01   Description

Project Overview

This project implements Amplitude Modulation (AM) and Frequency Modulation (FM) — both modulation and demodulation — using an Arduino microcontroller combined with physical electronic components and bench-top laboratory instruments.

For AM, the Arduino generates a carrier sine wave via PWM and mixes it with an audio-frequency message signal (produced by a function generator). The modulated signal is captured on an oscilloscope where the modulation index is measured. Demodulation is performed using an envelope detector circuit (diode + RC low-pass filter) whose output is displayed and compared to the original message.

For FM, the carrier frequency is varied proportionally to the instantaneous amplitude of the message signal. FM signal is generated using two signal generators, one creating the carrier frequency and the other one creating by modulating frequency. Signals are combined using a Local Oscillator chip. Once the signal is modulated, the signal is converted from analog signal to digital signal using Arduino C code to digitize the signal.

After the signal is transmitted to the receiver, the digital-to-analog circuit (DAC) is used to convert the signal back to analog, and then feed it back into the computer system for demodulation. Demodulation is performedusing a built-in arduino demodulation function, whose output is fed back to the Arduino now capture and output on both oscilloscope and on computer as readable values.

📶 AM — Amplitude Modulation

Carrier amplitude varies with message signal. Simple to implement; susceptible to noise. Modulation index μ = A_m / A_c determines sideband power distribution.

〰️ FM — Frequency Modulation

Carrier frequency deviates with message signal amplitude. Higher noise immunity than AM. Deviation ratio β = Δf / f_m determines bandwidth (Carson's Rule: BW ≈ 2(Δf + f_m)).

Key Outcomes

  • AM carrier envelope clearly visible on oscilloscope; modulation index measured from upper/lower envelopes within ±2% of target μ = 0.8.
  • AM demodulation via diode + RC envelope detector successfully recovered 1 kHz message signal with minimal distortion.
  • FM sidebands confirmed on spectrum analyzer matching Carson's Rule bandwidth prediction (BW ≈ 2 × (3 kHz + 1 kHz) = 8 kHz).
  • PLL demodulator reconstructed FM message signal with SNR exceeding 28 dB — superior to AM demodulator's 21 dB.
  • Python FFT spectrum analysis script successfully identified carrier frequency, sidebands, and harmonic distortion products.
  • Lab report documented full circuit schematics, measured vs. theoretical waveforms, and noise analysis for both modulation schemes.
