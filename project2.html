01   Description

Project Overview

The Delta-V Calculator is a Python-based orbital mechanics tool for computing the propulsive budget required to execute interplanetary and Earth-orbital transfers. Starting from Keplerian orbital elements, the tool calculates the total Δv for common maneuver sequences including Hohmann transfers, bi-elliptic transfers, plane changes, and gravity-assist flyby sequences.

The calculator supports both circular and elliptical departure/arrival orbits and accounts for the Oberth effect when burns occur inside gravity wells. Users can define multi-body mission profiles (e.g., Earth → Mars → asteroid) and receive a full Δv budget breakdown with recommended burn epoch windows.

Built primarily with NumPy and SciPy, it uses the vis-viva equation as its foundation and integrates Lambert's problem solver for arbitrary transfer arcs.

// Key equations
Δv = √(μ/r₁) × (√(2r₂/(r₁+r₂)) − 1)   [Hohmann departure burn]
Δv = √(μ/r₂) × (1 − √(2r₁/(r₁+r₂)))   [Hohmann arrival burn]
v² = μ × (2/r − 1/a)                     [Vis-viva equation]

03   Code / Implementation

Hohmann Transfer Solver

The core module computes departure and arrival Δv for a Hohmann transfer between two circular orbits in the same plane around a central body.

# delta_v.py — orbital transfer calculator
import numpy as np
from dataclasses import dataclass

# Gravitational parameters (km³/s²)
MU = {
    "Sun"  : 1.327124e11,
    "Earth": 3.986004e5,
    "Mars" : 4.28282e4,
}

@dataclass
class HohmannResult:
    dv1      : float   # Departure burn (km/s)
    dv2      : float   # Arrival burn  (km/s)
    dv_total : float   # Total budget  (km/s)
    tof      : float   # Transfer time (days)

def hohmann(r1: float, r2: float, body="Earth") -> HohmannResult:
    """
    Compute Hohmann transfer Δv between two circular orbits.
    r1, r2: radii in km from body centre.
    """
    mu = MU[body]
    a_transfer = (r1 + r2) / 2           # Semi-major axis of transfer ellipse

    v_circ1 = np.sqrt(mu / r1)            # Circular velocity at r1
    v_circ2 = np.sqrt(mu / r2)            # Circular velocity at r2
    v_per   = np.sqrt(mu * (2/r1 - 1/a_transfer))  # Speed at periapsis
    v_apo   = np.sqrt(mu * (2/r2 - 1/a_transfer))  # Speed at apoapsis

    dv1 = v_per - v_circ1
    dv2 = v_circ2 - v_apo
    tof = np.pi * np.sqrt(a_transfer**3 / mu) / 86400  # seconds → days

    return HohmannResult(
        dv1=round(dv1, 4),
        dv2=round(dv2, 4),
        dv_total=round(dv1 + dv2, 4),
        tof=round(tof, 2)
    )

# ── Example: LEO → GEO ───────────────────────────────────────
r_LEO = 6371 + 400   # km (400 km altitude)
r_GEO = 42164         # km
result = hohmann(r_LEO, r_GEO, body="Earth")
print(f"ΔV₁ = {result.dv1} km/s")
print(f"ΔV₂ = {result.dv2} km/s")
print(f"Total = {result.dv_total} km/s | TOF = {result.tof} days")
